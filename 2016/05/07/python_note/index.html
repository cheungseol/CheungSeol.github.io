第一行：以utf-8编码文件，可以显示中文 print u’中文’

第二行：从环境变量中的python解释执行程序，对于linux，当chmod a+x filename 后， 程序文件变为可执行文件， 可以用 ./ filename 执行, 如果在环境变量中加入程序文件名后，直接输入 filename 就可以执行代码文件

变量

使用变量时只需要给它们赋一个值。不需要声明或定义数据类型。 

i = 5
print i
i = i + 1
print i

while

There is a difference between them,

continue skips the loop’s current iteration and executes the next iteration.

pass does nothing. It’s an empty statement placeholder.

>>> for element in some_list:
... if element == 1:
... print "Pass executed"
... pass
... print element
... 
0
Pass executed
1
2

>>> for element in some_list:
... if element == 1:
... print "Continue executed"
... continue
... print element
... 
0
Continue executed
2
